# 🤖 CYBORG // CORE - Landing Page

Welcome to **CYBORG // CORE**, a futuristic, highly responsive, and neon-themed cyberpunk landing page. This project was developed as part of the **IIT Bombay Techfest College Ambassador** technical task.

---

## 🚀 Live Demo
🔗 **[Click Here to View the Live Project](https://cyborg-page.vercel.app)**

---

## 🎨 Design & Theme Concept
The website is built entirely around a **Cyborg (Human + Machine)** concept, blending biological elements with cybernetic enhancements.
* **Color Palette:** Dark-mode background (#0a0a0c) paired with high-contrast **Electric Cyan** and **Neon Magenta** accents.
* **Typography:** Futuristic headers using the *Orbitron* font and sharp, tech-oriented body text using *Rajdhani*.
* **Background Grid:** A custom CSS digital grid overlay mimicking a terminal or HUD environment.

## ✨ Features
* **100% Fully Responsive Layout:** Seamlessly adapts to desktops, tablets, and smartphones.
* **Cyberpunk UI Elements:** Smooth interactive glassmorphism navigation, neon text shadows (`text-shadow`), and glowing button borders.
* **Interactive Hover States:** Cards dynamically shift upwards and reveal custom animated cyber-borders when hovered.
* **Clean & Semantic Code:** Written in structured HTML5 and modern CSS3 (utilizing native CSS Variables and Grid layout).

---

## 🛠️ Tech Stack Used
* **HTML5:** For semantic web structure.
* **CSS3:** For custom styling, animations, layouts (Grid/Flexbox), and neon themes.
* **Google Fonts:** For importing specialized tech typography (*Orbitron* & *Rajdhani*).
